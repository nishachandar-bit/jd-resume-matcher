# JD-Resume-Matcher Streamlit App

This repository contains a Streamlit app that extracts skills from a Job Description and matches uploaded resumes against those skills.

To deploy:
1. Push this repo to GitHub.
2. On Streamlit Cloud (https://share.streamlit.io/) create a new app and point it to this GitHub repo.
3. Streamlit will build the app and provide a public URL.

Files:
- app.py: the Streamlit application
- requirements.txt: Python dependencies
